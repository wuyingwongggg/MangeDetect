Purpose

The purpose of this website is to provide the ability to upload images of wombats, in which the website will detect if the wombat has symptons of mange or not.

How does this work?

The website leverages PyTorch and ResNet, to create a custom model tailored to the classification of wombats. This is done by sampling data of wombats. Through this model, we hope to help discern between wombats with and without the symptons of sarcoptic mange.
